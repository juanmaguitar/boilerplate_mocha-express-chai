# Express + Mocha & Co

**Note**: `Grunt` is optional you can do all with `mocha` only

## Usage

```bash
npm test
```

or

```bash
mocha
```

or

```bash
grunt test
```

## Autotest

```
mocha -w
```
or

```
grunt watch
```

## Coverage

```
mocha -R html-cov > coverage.html
```

## Test Documentation

```
mocha -R doc > doc.html
```
